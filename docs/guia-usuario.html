03 Crear un Nuevo Producto

Campos del formulario

  • Nombre — nombre completo del repuesto (requerido)
  • Categoría — seleccioná de la lista (requerido)
  • Subcategoría — refinamiento dentro de la categoría
  • Marca del repuesto — Brembo, NGK, Bosch, etc.
  • Precio — precio actual en pesos (puede dejarse vacío)
  • Precio anterior — para mostrar tachado
  • Badge — etiqueta: Nuevo, Oferta, Premium, etc.
  • Descripción — texto corto opcional
  • Stock — activado/desactivado + cantidad exacta
  • Destacado — aparece en la home si está tildado

Subir fotos del producto

  • Hacé clic en + Agregar foto o arrastrá una imagen al área
  • Se pueden subir múltiples fotos por producto
  • Formatos: JPG, PNG, WebP (máx. 5 MB por imagen)
  • La primera foto es la imagen principal del catálogo
  • Las fotos se guardan en Supabase Storage automáticamente
  • En el catálogo, el cliente puede navegar entre fotos con flechas ←→
Watermark automática: cada imagen muestra "DANICHAP" en la esquina inferior derecha para proteger el contenido.
⚠️ Importante: El nombre del producto debe ser claro y descriptivo. Evitá nombres como "Producto nuevo" o "ada (copia)". El nombre aparece en el catálogo público y en los mensajes de WhatsApp.
04 Editar, Duplicar y Eliminar Productos
Acción Ícono Qué hace Cuándo usarlo
Editar ✏️ lápiz Abre el formulario con los datos actuales del producto. Permite cambiar nombre, precio, fotos, stock, etc. Para actualizar precio, subir nuevas fotos o corregir información.
Duplicar 📋 copia Crea una copia del producto con "(copia)" en el nombre. La copia queda en modo edición. Para crear variantes del mismo producto rápidamente.
Eliminar 🗑️ tacho Elimina el producto permanentemente de la base de datos y sus imágenes del storage. Solo cuando el producto ya no se vende y no querés que aparezca más.
Destacar ⭐ estrella Marca o desmarca el producto como destacado. Los destacados aparecen en la home. Para promover productos con mayor visibilidad en la página principal.

05 Control de Stock y Etiquetas (Badges)

Control de stock

  • Stock activado: el producto aparece con estado disponible
  • Stock desactivado: el producto queda oculto/sin disponibilidad
  • Cantidad exacta: si ponés un número ≤5, el catálogo muestra el aviso "Últimas N uds." en naranja para generar urgencia
  • Útil para productos con pocas unidades disponibles

Etiquetas (Badges) disponibles

Nuevo Oferta Premium Stock Limitado Original

Las etiquetas aparecen en la esquina superior derecha de la imagen del producto en el catálogo. Son opcionales — podés dejar el campo vacío si no querés badge.

06 Cómo Funciona el Catálogo Público
Origen de los productos: El catálogo muestra los productos de la base de datos (admin) + una base de datos estática. Los de la BD tienen prioridad. Los cambios en el admin se reflejan en el catálogo en tiempo real.

🔍 Búsqueda de texto

El cliente escribe en la barra de búsqueda. El filtro busca en nombre, marca, descripción y categoría en tiempo real. No hace falta presionar Enter.

🚗 Filtro por vehículo

El cliente selecciona Marca → Modelo → Versión → Año. El sistema filtra solo los repuestos compatibles con ese vehículo. La compatibilidad se define en el código (data.js).

🗂️ Filtro por categoría

El sidebar izquierdo muestra las categorías disponibles. Al hacer clic se filtra automáticamente. Cada categoría tiene subcategorías expandibles.

📷 Galería de imágenes

Si el producto tiene múltiples fotos, el cliente puede navegar entre ellas con las flechas ← → dentro de la tarjeta, o haciendo clic en la imagen para abrir el detalle completo con todas las fotos y descripción.

📊 Ordenamiento

El cliente puede ordenar los resultados por: Relevancia (default), Precio: menor a mayor, Precio: mayor a menor, Nombre A-Z. El ordenador está en la barra superior del catálogo.

Mensaje automático generado

Al hacer "Enviar pedido por WhatsApp", se genera un mensaje con todos los productos seleccionados, cantidades, y el vehículo del cliente si usó el filtro. El cliente solo debe enviarlo.

Ejemplo: "Hola Danichap! Consulto por: *Pastillas Brembo* para mi vehículo *Toyota Corolla 2018*. ¿Tienen stock y precio?"

Consulta directa (sin carrito)

Cada tarjeta tiene un botón verde "Consultar" que abre WhatsApp directamente con el nombre del producto y vehículo. No requiere iniciar sesión ni usar el carrito.

El FAB (botón flotante verde ↗) en la esquina inferior derecha permite contacto directo en cualquier momento.

08 Buenas Prácticas para el Admin

✅ Nombres claros

Usar nombres completos y descriptivos. Ej: "Pastillas de freno Brembo P85020" en lugar de "freno".

✅ Fotos de calidad

Subir fotos sobre fondo blanco o neutro, bien iluminadas. Mínimo 600×600px. El watermark DANICHAP se agrega automáticamente.

✅ Precio actualizado

Si el precio cambió, actualizar en el admin. Si no hay precio fijo, dejar vacío para que aparezca "Consultar por WhatsApp".

✅ Usar categorías correctas

Asignar siempre la categoría correcta para que el filtro del catálogo funcione bien.

✅ Stock al día

Desactivar stock de productos agotados. Si queda poco, poner la cantidad exacta para mostrar urgencia al cliente.

⚠️ No dejar copias

Al duplicar un producto, editarlo o eliminarlo si no se usa. Evitar nombres como "ada (copia) (copia)" en el catálogo público.
